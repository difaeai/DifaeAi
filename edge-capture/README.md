# Edge Capture Toolkit

Supporting assets for capturing vendor-locked P2P cameras and forwarding them into the bridge.

- `windows/` – Portable OBS setup (`CameraBridge-OBS.zip`) plus launcher script.
- `linux/` – `scrcpy` + `ffmpeg` helper script for Android-based capture.

Follow the playbooks in `docs/camera-bridge/p2p-windows.md` and `p2p-android.md` for detailed steps.
